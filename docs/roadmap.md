# Product Roadmap

## Phase 1 – MVP

Goal:

Validate demand for simplified meal nutrition calculation.

Features:

* Food search
* USDA integration
* Quantity input
* Meal builder
* Nutrition calculation
* Meal summary

Success Criteria:

Users can calculate meal nutrition in under one minute.

---

## Phase 2 – Convenience Features

Goal:

Reduce repetitive meal entry.

Features:

* Saved meals
* Recent searches
* Favorite foods
* Daily nutrition dashboard

---

## Phase 3 – AI-Assisted Experience

Goal:

Reduce manual input.

Features:

* Natural language meal entry

Example:

"50g oats, 2 scoops whey, 180ml milk"

* Ingredient extraction using LLM

---

## Phase 4 – Progress Tracking

Goal:

Connect nutrition behavior to outcomes.

Features:

* Weight tracking
* Goal tracking
* Weekly summaries
* Progress insights

---

## Phase 5 – Personalization

Goal:

Provide actionable recommendations.

Features:

* Personalized calorie targets
* Protein recommendations
* Nutrition insights
* Goal-based suggestions
